Виконай редагування існуючого UI платформи для програмування CodeArena. Це не новий дизайн з нуля, а внесення змін і покращень у поточний інтерфейс із збереженням стилю (темна тема, фіолетові акценти, glow-ефекти, мінімалізм, сучасний UI).

🔧 1. Зміни в існуючому інтерфейсі

Кнопка "Приєднатися":

Зроби кнопку в темному стилі (dark button)
Текст кнопки — в основному акцентному кольорі (наприклад фіолетовий)
Додай легкий glow / hover-ефект

Navbar (верхня навігація):

Біля назви "CodeArena" додай 2 вкладки:
Огляд (основна сторінка з турнірами)
Турніри
У вкладці Турніри:
Залиш тільки кнопку "Мої турніри"
Видали всі інші кнопки

Меню профілю (dropdown при натисканні на аватар):

Видали кнопку "Мої турніри"
Додай перемикач мови:
UA / EN
При перемиканні змінюється весь інтерфейс сайту на англійську

Карточки турнірів:

Біля опису кожного турніру додай маленький тег теми:
Наприклад: "Graphs", "DP", "Math"
Стиль: маленький badge, округлений, з легким фоном
🪟 2. Модальне вікно "Список задач"

Створи модальне вікно, яке відкривається при натисканні кнопки "Приєднатися".

Стиль:

Повністю відповідає модальному вікну "Зареєструватися"
Темна тема
Заокруглені кути
Сучасний мінімалістичний UI
Легка тінь + backdrop blur

Контент модального вікна:

Заголовок:

"Список задач"

Основна частина:

Вертикальний список задач
Формат: рядки (не карточки)

Кожен рядок містить:

Літерний індекс (A, B, C, D...)
Назву задачі

Взаємодія:

Hover ефект (зміна фону / підсвітка)
При натисканні:
Перехід на сторінку задачі (як зараз працює кнопка "Приєднатися")

Нижня частина модалки:

Блок з коротким описом турніру (1–2 абзаци)
Менш акцентований текст (lower opacity / secondary text)

Додатково:

Кнопка закриття (хрестик у правому верхньому куті)
Scroll якщо багато задач
Всі відступи, шрифти і spacing як у модальному вікні реєстрації
🧾 3. Реєстрація користувача

Navbar:

Додай кнопку "Зареєструватися" справа (біля "Team")
Кнопка:
Адаптивна (flex)
Підлаштовується під тему
Має фіолетові акценти + glow ефект

Вікно реєстрації:

Відкривається при натисканні кнопки
Стиль:
Відповідає поточній темі
Мінімалістичний
Акуратний і сучасний

Поля:

ПІБ (повне ім’я)
Логін
Пароль
Email
Дата народження
Школа (опціонально)

Кнопка:

"Зареєструватися"
Такий самий стиль як у navbar

Адаптивність:

Вікно коректно виглядає на різних розмірах екранів
Підлаштовується під зміну теми
🎨 Загальні вимоги
Зберегти існуючий дизайн системи
Не ламати структуру
Підтримувати:
темну тему
фіолетові акценти
glow ефекти
Всі нові елементи повинні виглядати нативно в UI
Зберегти карточний стиль турнірів
Дотримуватись мінімалізму і візуального балансу
