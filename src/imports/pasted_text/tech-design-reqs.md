⚙️ ТЕХНІЧНІ ВИМОГИ:
- Використовуй підхід React для структури компонентів
- Де можливо — застосовуй TypeScript
- Компонентна архітектура
- Чисті та повторно використовувані UI-компоненти
- Готовність до передачі розробникам (auto-layout, constraints)

📝 МОДАЛЬНЕ ВІКНО РЕЄСТРАЦІЇ:

СТИЛЬ:
- Квадратна картка
- Великі заокруглення
- Glow-ефект
- Темна тема

КОНТЕНТ:

Заголовок:
"Реєстрація команди"

Опис:
"Команда може самостійно зареєструватися між датами реєстрації."

ФОРМА:

- Назва команди (обов’язково)
- Капітан:
  - ПІБ
  - Email
- Учасники:
  - Динамічний список
  - ПІБ + Email
  - Мінімум 2
  - Максимум N (налаштовується)
- Місто / школа / організація (опціонально)
- Telegram / Discord (опціонально)

ВАЛІДАЦІЇ:

- Email унікальні в межах команди
- Захист від дублювання:
  - той самий капітан або ті самі email

ПІСЛЯ РЕЄСТРАЦІЇ:

- Редагування складу:
  - Заборонено
  - Опціонально: лише для Admin
